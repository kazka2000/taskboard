# 프로젝트 기능 명세서

이 문서는 Taskboard 애플리케이션의 현재 구현된 기능들을 사용자 역할과 기능별로 정리한 내용입니다.

## 1. 사용자 역할 (User Roles)

### 일반 멤버 (General Member)
- **접근 권한**: 로그인 및 대시보드 조회 가능.
- **주요 기능**:
  - 본인이 소유하거나 할당된 프로젝트 관리.
  - 해당 프로젝트 내에서 태스크 생성 및 관리.
  - 댓글을 통한 협업 참여.

### 관리자 (Administrator)
- **접근 권한**: 시스템 전체 접근 및 관리자 패널(Admin Panel) 이용 가능.
- **주요 기능**:
  - 일반 멤버의 모든 권한 포함.
  - **글로벌 접근**: 멤버십 여부와 관계없이 시스템 내 **모든** 프로젝트 조회 및 관리 가능.
  - **사용자 관리**: 신규 사용자 생성 및 역할(관리자/멤버) 부여, 사용자 목록 조회.

---

## 2. 기능별 상세 (Features by Functionality)

### 🔐 인증 및 보안 (Authentication)
- **로그인 시스템**: 아이디(Username)와 비밀번호를 이용한 보안 로그인.
- **세션 관리**: 사용자 로그인 상태 유지 (세션 지속성).

### 📋 대시보드 및 프로젝트 관리
- **프로젝트 생성**: 
  - 제목, 설명, 기간 설정과 함께 새 프로젝트 생성.
  - **템플릿 지원**: "기본(Basic)", "웹 개발(Web Development)", "영업(Sales)" 등 프리셋을 선택하여 관련 컬럼 자동 생성.
- **프로젝트 관리**:
  - 프로젝트 정보 수정 (제목, 설명, 기간).
  - 프로젝트 삭제 (관련된 모든 리스트와 태스크 포함).

### 📊 칸반 보드 시스템
- **리스트/컬럼 관리**:
  - 새로운 커스텀 컬럼 생성.
  - **드래그 앤 드롭 정렬**: 마우스 드래그로 컬럼 순서 변경.
  - 기존 컬럼 이름 변경 및 삭제.
- **태스크 시각화**: 각 태스크를 컬럼 내 카드 형태로 시각적으로 배치.

### ✅ 태스크(작업) 관리
- **CRUD 작업**: 태스크의 생성(Create), 조회(Read), 수정(Update), 삭제(Delete) 완벽 지원.
- **태스크 상세 정보**:
  - **기본 정보**: 제목, 상세 설명(Rich Text), 태그(일반, 긴급, 버그, 기능, 디자인 등).
  - **일정 관리**: 마감일(Deadline/Due Date) 설정.
  - **담당자 배정**: 하나의 태스크에 다수의 담당자 배정 가능 (아바타 표시).
- **인터랙션**:
  - **드래그 앤 드롭**: 컬럼 간 태스크 이동(상태 변경) 및 컬럼 내 순서 재정렬.
  - **클릭하여 편집**: 상세 모달창을 통해 고급 편집 기능 제공.

### 💬 협업 기능 (태스크 상세 뷰)
- **댓글(Comments)**: 특정 태스크에 대해 텍스트 댓글 작성 및 공유.
- **활동 로그(Activity Log)**: 주요 작업에 대한 시스템 자동 로그 기록:
  - 리스트 간 태스크 이동 내역.
  - 주요 정보 업데이트(제목, 설명 등) 내역.
  - **참고**: 누가, 언제 변경했는지 추적 가능하여 협업 효율 증대.

### ⚙️ 시스템 및 관리자 기능
- **사용자 관리 (관리자 전용)**:
  - 새로운 사용자 추가 (ID, 역할 지정).
  - 전체 등록된 사용자 목록 및 역할/ID 조회 테이블 제공.
- **다국어 지원 (Localization)**:
  - **언어 전환**: 헤더 메뉴를 통해 **한국어**와 **영어** 간 실시간 전환 지원.
- **타임존(Timezone)**:
  - **KST (한국 표준시)**: 서버 및 데이터베이스가 한국 표준시(GMT+9)로 설정되어 정확한 마감일 관리 보장.
